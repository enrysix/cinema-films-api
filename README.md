# cinema-films-api

## Common Setup

1. Clone the repo and install dependencies

```bash
git clone https://github.com/enrysix/cinema-films-api.git
cd cinema-films-api
```

2. Install dependecies

```bash
npm i
```

### Compiles and hot-reloads for development

```
npm run serve
```

Server serviced on http://localhost:3000
